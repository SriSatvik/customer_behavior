# Data Analysis Project

## Overview

This project demonstrates an end-to-end data analysis workflow, from loading and cleaning raw data to exploring insights, querying data with SQL, and creating interactive dashboards.

The project focuses on transforming raw data into meaningful business insights using Python, PostgreSQL, and Power BI.


## Dataset

The dataset contains structured data used for analysis and visualization.

Key steps performed:

* Loaded data using Python
* Cleaned missing and duplicate values
* Corrected data types
* Prepared data for analysis


## Tools & Technologies

* **Python** – Data loading and preprocessing
* **Pandas** – Data cleaning and manipulation
* **NumPy** – Numerical operations
* **PostgreSQL** – SQL queries and data analysis
* **Power BI** – Interactive dashboards and visualizations

---

## Project Workflow

1. Load the dataset using Python.
2. Clean and preprocess the data.
3. Perform Exploratory Data Analysis (EDA).
4. Import cleaned data into PostgreSQL.
5. Write SQL queries to answer business questions.
6. Build interactive Power BI dashboards to present insights.

## Exploratory Data Analysis (EDA)

The EDA includes:

* Summary statistics
* Missing value analysis
* Distribution of key variables
* Correlation analysis
* Trend analysis
* Category-wise comparisons



## SQL Analysis

SQL was used in PostgreSQL to:

* Filter and aggregate data
* Perform joins
* Calculate KPIs
* Analyze trends
* Answer business-related questions


## Dashboard Results

The Power BI dashboard provides:

* Interactive filters and slicers
* KPI summary cards
* Trend analysis
* Category-wise performance
* Visual insights for decision-making

  
## Key Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis (EDA)
* SQL Query Writing
* Data Visualization
* Dashboard Development
* Business Insight Generation


## Conclusion

This project showcases a complete data analytics workflow using Python, PostgreSQL, and Power BI. It highlights practical skills in data preparation, analysis, SQL querying, and dashboard development to generate actionable insights.
